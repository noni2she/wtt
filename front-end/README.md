# Client Server

## Basics
### Prerequisite
* node >= 6.0.0

## Getting started

```sh
npm install

npm run dev
```

navigate to ```http://localhost:8080``` in your browser.

### Dev with real-time data 
```sh
npm run dev:real-data
```
